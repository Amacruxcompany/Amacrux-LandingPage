The universal NFT marketplace or Web 3.0 starts here.

Amacrux is an open source decentralized marketplace that provides practical modules for minting non-fungible tokens (NFTs) represented as useful digital assets in the real and virtual world, allowing buyers and sellers to interact with collectibles, apply to online stores and participate in crowdfunding rounds.

MARKETPLACE DEMO

A universal open and decentralized marketplace.

Amacrux is a universal open-source marketplace supported by developers, creators and collectors interested in trustless peer-to-peer trading of digital assets, through an interface that provides dynamic tools for easier and more comfortable exploration, preparation and launch of NFTs with real-world utility.

SocialFi model with directional benefits

The SocialFi model adds value to NFT properties by driving user-to-user interactions by tokenizing social activities to influence the quality of content (asset). Within the marketplace, each user and NFT contract address (URL link) will be assigned a certain influence score that is recorded by Amacrux's native algorithm to approve future rewards for content with higher social value. This algorithm will also be used to test the preparation stages of projects that pass the minimum requirements to qualify for crowdfunding.

Automated rewards distribution

Users can generate royalty statements for secondary sales with an adjustable range between 0 and 10% of the final amount of the NFT to be traded. Amacrux royalties present an opportunity to monetize the uniqueness of the asset expressed by authors, allowing them to earn passive income from each transaction between third parties.

Create NFTs for free using delay minting function

In Amacrux, users will be able to create NFTs without listing services cost using the "delay minting" function. Delay minting guarantees the integrity of the NFTs listed in the market, storing the data in a decentralized and safe way on IPFS.

Community governance powered by DAO

Amacrux proposes a democratic government model capable of giving up formal control to market players such as creators, collectors and other interested parties. As a result, the marketplace will become a decentralized autonomous organization (DAO) with the authority to implement contract changes based on assigned proposals in official repositories (open-source principle).
